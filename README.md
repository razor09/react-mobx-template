### Установка и запуск

```
npm ci

npm start
```

### Локальный запуск production сборки по адресу `http://localhost:4200`

```
npm run build

npm run serve
```

### Линтер

```
npm run lint
```

### Кодогенерация

```
npm run generate:component MyFirstComponent MySecondComponent

npm run generate:store MyFirstStore MySecondStore

npm run generate:mocks myFirstMocks mySecondMocks
```
